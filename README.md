# Calculadora de Cultivo de Cannabis para REPROCANN

Calculadora gratuita para planificar el cultivo de cannabis medicinal según los objetivos de tratamiento. Parte de la dosis diaria necesaria y calcula hacia atrás cuántas plantas, cuántos ciclos por año y qué producción hacen falta para cubrirla.

Herramienta desarrollada para pacientes, clubes de cultivo y asociaciones civiles inscriptas en el **REPROCANN**, como apoyo a la elaboración del Plan de Cultivo.

## ¿Qué hace esta calculadora?

- Estima la **cantidad de plantas** necesarias según la dosis diaria (en mg/día o en gotas y concentración).
- Calcula los **ciclos de cultivo** por año y la producción esperada.
- Orienta sobre superficie y tamaño de maceta.
- Funciona para cultivo indoor y outdoor.
